Hospital Appointment System

Requirements:
- JDK 17+
- Apache Tomcat
- MySQL
- VS Code Java Extensions

How to Run:
1. Import database.sql into MySQL.
2. Open project folder in VS Code.
3. Add MySQL Connector JAR.
4. Configure Tomcat Server.
5. Run the project.
